python模块之glob

Posted on August 24, 2018 本文总阅读量次

python之glob模块那点事儿

glob的背景

glob的匹配符

glob查找文件只用到3个匹配符号 * ？ [] 
* 代表 0个或者多个字符
？代表1个字符
[] 代表的是查找的范围 如[0-9]匹配数字

glob模块的方法

1.glob.glob()

该方法返回所有匹配的文件路径列表。该方法需要一个参数用来指定需要匹配的路径字符串（字符串可以为绝对路径也可以为相对路径）其中返回的文件路径列表只会返回当前路径中匹配到的。还是先上一段代码：

import glob
file=glob.glob(r'*.py') #列出当前路径所匹配的所有的.py的文件 
file1=glob.glob(r'C:\Users\admin\Pictures\Saved Pictures\*.jpg')  # 这里是去绝对路径里面去寻找
## 上面的返回都是一个list

2.glob.iglob()

获取一个迭代器（iterator ）对象，使用它可以逐个获取匹配的文件路径名。与glob.glob()的区别是一次只获取一个文件路径名。而glob.glob()一次获取所有的匹配的文件名。接着上代码

  import glob
  filelist=glob.iglob(r'*.py')
  print filelist # <generator object iglob at 0x02DA9FD0>
  for file in filelist:
      print file       #一次只返回一个
